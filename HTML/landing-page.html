SISTEM INFORMASI TUGAS AKHIR

Sistem Informasi Tugas Akhir, merupakan aplikasi mobile yang dirancang untuk mempermudah proses pengerjaan tugas akhir mahasiswa informatika. Sistem ini menyediakan akses mudah ke informasi penting dan melancarkan komunikasi antara mahasiswa dengan dosen pembimbing.

FITUR

Dalam Sebuah pengembangan sistem membutuhkan fitur-fitur tertentu. adapaun fitur dalam aplikasi ini yaitu

Daftar Dosen

Aplikasi SITA merancang untuk User bisa melihat kontak dosen dan mempermudah komunikasi.

Tugas Akhir

Aplikasi SITA merancang agar mempermudah User dan Admin dalam mengelola Tugas Akhir.

Jadwal Sidang

Aplikasi SITA merancang agar mempermudah User dan Admin dalam melihat Daftar Sidang.

Logbook

Aplikasi SITA merancang agar mempermudah User dan Admin dalam melihat catatan Logbook..

SITA Website

Aplikasi Sistem Informasi Layanan Tugas akhir Jurusan Teknik Informatika sudah ada dalam bentuk Website dan bisa di akses.

SITA Mobile

Aplikasi Sistem Informasi Layanan Tugas akhir Jurusan Teknik Informatika Berbasis Mobile.Sebenarnya sudah ada dalam bentuk Website dan bisa di akses Tetapi Project kali ini, kami mengembangkan lagi dalam bentuk Mobile dan juga Fitur tambahan yang mungkin nanti seiring berjalannya waktu akan ada Fitur baru. Jadi bisa disimpulkan bahwasanya ini bukan merupakan Project Lanjutan, melainkan Pengembangan Project dari bentuk Website menjadi Mobile.
